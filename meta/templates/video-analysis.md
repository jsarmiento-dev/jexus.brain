creator: ""
url: ""
duration: ""
date_analyzed: ""
tags: []
complexity: "" # Básico | Intermedio | Avanzado
topics: []
---

---

## 📋 Índice rápido por campos

| Campo | Descripción |
| :--- | :--- |
| **Título** | Nombre exacto del video |
| **Categoría** | Sub-tema (ej: LLMs, Agentes, Automatización, Herramientas) |
| **Duración** | Tiempo total [HH:MM:SS] |
| **Sinopsis** | Resumen ejecutivo (3-5 líneas) con conceptos clave |
| **Enlace** | URL directa al video |
| **Complejidad** | Básico / Intermedio / Avanzado |
